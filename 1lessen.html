Редактирования текста.

Чтобы сделать заголовок и подзаголовок я использую теги:

  1. <h1> Заголовок </h1>
  2. <h2> Заголовок </h2>
  3. <h3> Заголовок </h3>
  4. <h4> Заголовок </h4>
  5. <h5> Заголовок </h5>

Цвет текста можно сделать и в HTML с помощью атрибута color.

<font color="blue(Любой Другой Цвет на АНГЛ)"> Текст </font>

Так же можно установить цвет с помощью CSS.

<p class="page-header"> Текст </p> 

            <style>
            .page-header {
                color: #ff0000;
            }
        </style>
